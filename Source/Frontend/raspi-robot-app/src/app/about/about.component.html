RaspiRobot is a Raspberry PI based 6-axis robot that can be controlled by any EROWA Robot OpenAPI compatible process control like JMS 4.0 ProductionLine.

It provides several interfaces:

  • EROWA Robot OpenAPI: A gRPC based interface to command and observe the robot by a process control system
  • WebAPI used by this Angular based frontend
  • Swagger for developers (or otherwise interested people) to try out the WebAPI. RaspiRobot's Swagger page is accessible here.
